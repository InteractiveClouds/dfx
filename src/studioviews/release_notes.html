What's New?

This version is the first public release of DreamFace in its version 3. A lot of changes have been made on the UI, by including Angular Material as the core framework for Graphical Controls. This specific build has the following fixes & new features:

  • New service to call an API route from a view: dfxApiServices (IMPORTANT: this new service replaces dfxApiRoutes, you have to update manually your controllers).
  • API Service filters new syntax: now the complete function signature is accessible (preExecutionFilter() / postExecutionFilter())
  • Progress bar Graphical Control
  • Carousel bar Graphical Control
  • 2 new attributes on layout columns: Auto Width and Display

What's coming?

The upcoming features will be released soon:

  • Manage Environments & global variables
  • Personalization capabilities per role
  • Flex support on Views in pages
  • Configurable Login page
  • Graphical Control Templates
