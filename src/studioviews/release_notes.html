What's New?

This version contains major fixes. The list is displayed below:

  • Add input text (or select or multitext or date) , change their size to 80% for example. Save, quit, re-open: size is back to its original size (20 for input, 50 for multi line), etc. Same happens with existing Views where such components are being used (they all look with their original sizes)
  • Warnings generated by console on use of the select GC
  • Select GC expression picker does not open
  • Select GC bound to dynamic options error
  • Copy view from one category to another

Enhancements and new features:
